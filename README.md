# Trade Opportunities API

A FastAPI service that analyzes market data and provides trade opportunity insights for specific sectors in India. Powered by **Google Gemini AI** and real-time web search via **DuckDuckGo**.

## Features

- **Single endpoint** — `GET /analyze/{sector}` returns a structured markdown report
- **AI-powered analysis** — Uses Google Gemini to generate professional market reports
- **Live data** — Searches current market news via DuckDuckGo
- **Authentication** — API key based (`X-API-Key` header)
- **Rate limiting** — Token bucket algorithm (per IP)
- **Session tracking** — Tracks usage per client IP
- **Input validation** — Regex-based sector name validation
- **In-memory storage** — No database required

## Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/ragbendra/Appscrip-Raghabendra_Python.git
cd Appscrip-Raghabendra_Python
```

### 2. Create a virtual environment

```bash
python -m venv venv
```

Activate it:

- **Windows**: `venv\Scripts\activate`
- **macOS/Linux**: `source venv/bin/activate`

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Configure environment variables

Copy the example file and fill in your keys:

```bash
cp .env.example .env
```

Edit `.env`:

```env
GEMINI_API_KEY=your_gemini_api_key_here
GEMINI_MODEL=gemini-2.0-flash
API_SECRET_KEY=your_secret_api_key_here
```

| Variable | Description |
|----------|-------------|
| `GEMINI_API_KEY` | Google Gemini API key ([get one free](https://aistudio.google.com/apikey)) |
| `GEMINI_MODEL` | Gemini model name (e.g. `gemini-2.0-flash`) |
| `API_SECRET_KEY` | Secret key for API authentication |

### 5. Run the server

```bash
uvicorn app.main:app --reload
```

The server starts at `http://127.0.0.1:8000`.

## API Usage

### Endpoint

```
GET /analyze/{sector}
```

### Sample Request

```bash
curl -H "X-API-Key: your_secret_api_key_here" http://127.0.0.1:8000/analyze/pharmaceuticals
```

### Sample Response

```json
{
  "sector": "pharmaceuticals",
  "generated_at": "2026-03-31T00:00:00Z",
  "report": "# Pharmaceuticals Sector — India Trade Opportunities Report\n\n## 1. Sector Overview\n..."
}
```

The `report` field contains a structured markdown document with the following sections:

1. **Sector Overview**
2. **Key Export Opportunities**
3. **Key Import Opportunities**
4. **Market Drivers**
5. **Key Players**
6. **Risks and Challenges**
7. **Analyst Outlook**

### Error Responses

| Status | Description |
|--------|-------------|
| `401` | Missing `X-API-Key` header |
| `403` | Invalid API key |
| `422` | Invalid sector name |
| `429` | Rate limit exceeded (check `Retry-After` header) |
| `500` | AI analysis failed |

## Security

- **Authentication**: API key verified on every request via `X-API-Key` header
- **Rate limiting**: Token bucket algorithm — 10 requests burst, 1 token/sec refill (configurable)
- **Input validation**: Sector names restricted to 2–50 characters (letters, spaces, hyphens only)
